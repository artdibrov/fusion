
# Image Gallery

A lightweight Python application for organizing photos from a smartphone and a desktop computer.

## Overview

Image Gallery combines photos from multiple devices into one collection. It can detect duplicate filenames, organize images by source device, and create simple virtual albums.

The project demonstrates modular programming, dictionaries, lists, and basic collection management.

## Features

- Combine smartphone and desktop libraries
- Group images by device
- Detect duplicate filenames
- Create virtual albums
- Display collection summary
